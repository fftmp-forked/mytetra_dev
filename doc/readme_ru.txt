
MyTetra v.1.x.x

MyTetra - менеджер накопления информации (PIM)


Установка портабельной версии MyTetra:
--------------------------------------

1. Создайте любую директорию
2. Скопируйте все файлы из архива в данную директорию
3. Запустите файл mytetra.run


Первый запуск портабельной версии MyTetra:
------------------------------------------

В момент первого запуска MyTetra появится диалог, в котором надо выбрать
режим установки программы: "Standard" или "Portable". Выберите "Portable"
и нажмите OK

В портабельном режиме MyTetra создаст файлы и директории, необходимые для работы
приложения прямо в каталоге с бинарником.

ВНИМАНИЕ! Если в момент установки вы выбрали стандартный режим (Standard),
то чтобы запустить MyTetra в портабельном режиме надо удалить директорию
"/.config/mytetra" в домашней директории пользователя. После этого
MyTetra запустится так, как будто запускается в первый раз.


Обновление версии программы
---------------------------

Для обновления с предыдущей версии, просто замените следующие файлы и директории:

iconengines  (directory)
imageformats (directory)
lib          (directory)
platforms    (directory)
mytetra      (binary file)
mytetra.run  (script file)

Новая версия сама подхватит существующие конфигурирующие файлы и базу данных.
